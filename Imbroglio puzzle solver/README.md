O solucionador funciona em 2 etapas:

O primeiro script encontra todas as palavras que é possivel formar com as letras de cada dia dentro da base de words.json

O segundo script encontra as combinações de palavras que utilizam todas as letras, ordenam pelo ponto e retorna as top 20 soluçõoes

O script "remove_words" remove do json "words.json" as palavras que estão em "words_to_remove.json"
